# Clasificación de correo electrónico no deseado mediante aprendizaje automático

Este repositorio contiene la implementación y los resultados experimentales de un sistema para la clasificación automática de correo electrónico no deseado mediante técnicas de procesamiento del lenguaje natural (NLP) y aprendizaje automático.

## Tecnologías
- Python
- Google Colab
- Scikit-learn
- Pandas
- NumPy
- Matplotlib

## Modelos implementados
- Naive Bayes
- Máquina de vectores de soporte (SVM)
- Bosque aleatorio

## Conjunto de datos
Conjunto de datos de recopilación de spam SMS (Kaggle)

## Resultados
La evaluación se realizó utilizando las siguientes métricas:
- Exactitud
- Precisión
- Recuperación
- Puntuación F1
- AUC-ROC

## Reproducibilidad
Todo el código y los resultados están disponibles públicamente para garantizar la reproducibilidad y la transparencia.
